# 单词正字法

何为「单词正字」？

在中文中，如果遇到不认识的生字，通过偏旁、部首能大致判断读音，如「蓓」，也许你不认识，但你会知道应该读「bei」或「pei」，英文也有类似规律，即「orthography」。本仓库目的是找出英文高频单词中的普遍正字规律，规则主要参考 [English orthography - Wikipedia](https://en.wikipedia.org/wiki/English_orthography)。
